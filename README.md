📄 Descrição do Projeto

Este projeto é uma aplicação web simples feita com HTML, CSS e JavaScript. Ele permite que o usuário digite seu nome, idade e também os dados de outra pessoa diretamente na tela.
Após preencher os campos e clicar no botão, o sistema coleta essas informações e organiza os dados para exibição na página.
Os dados são armazenados em uma lista (array) e também em um conjunto (Set), que serve para mostrar os valores sem repetição.

🚀 Funcionalidades
Inserir nome e idade em campos na tela
Coletar os dados ao clicar no botão
Validar se os campos foram preenchidos corretamente
Armazenar os dados em uma lista
Remover valores repetidos usando Set
Mostrar os dados na tela sem recarregar a página

🛠️ Tecnologias Utilizadas
HTML → estrutura da página
CSS → estilo e aparência
JavaScript → lógica e funcionamento

🎯 Objetivo
O objetivo deste projeto é praticar conceitos básicos de programação web, como:

Capturar dados do usuário
Trabalhar com listas (arrays)
Utilizar Set para evitar valores repetidos
Manipular elementos da página com JavaScript

💡 Observação
Este é um projeto simples, voltado para aprendizado, ideal para quem está começando no desenvolvimento web.
