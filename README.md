🧑‍💻 Cadastro de Pessoas (Array + Set) 🧑‍💻

📌 Descrição do Projeto
Este projeto é uma aplicação web simples desenvolvida com HTML, CSS e JavaScript.
A aplicação permite que o usuário digite nome e idade, além dos dados de outra pessoa, diretamente nos campos da tela. Após clicar no botão, as informações são coletadas e exibidas automaticamente na página, sem precisar recarregar.
Os dados são armazenados em uma lista (Array) e também em um Set, que é utilizado para remover valores repetidos e garantir que os dados exibidos não estejam duplicados.

🎯 Objetivo
O objetivo deste projeto é praticar conceitos básicos do desenvolvimento web, como:

Estruturação de páginas com HTML
Estilização com CSS
Captura de dados usando inputs
Manipulação do DOM com JavaScript
Armazenamento de dados em Arrays
Uso de Set para remover valores repetidos
Exibição dinâmica dos resultados na tela

✅ Funcionalidades

📌 O sistema permite:
Inserir nome e idade em campos na tela
Coletar os dados ao clicar no botão
Validar se os campos foram preenchidos corretamente
Armazenar os dados em um Array (lista)
Remover valores repetidos utilizando Set
Exibir os dados na tela automaticamente

🛠 Tecnologias Utilizadas
HTML5
CSS3
JavaScript
